## Backend Setup
These are the commands to run to configure your development machine before starting the backend code.

```shell
sudo pip3 install virtualenv
cd Backend/src
virtualenv venv
. venv/bin/activate
pip3 install -r requirements.txt
```

To start the server

```shell
cd Backend/src
python3 routes.py
```
